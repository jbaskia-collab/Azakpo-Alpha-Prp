# Azakpo-Signal-Pro
Build Alpha Analytics Pro, a dark-mode SPA using Tailwind and Firebase. ​Auth: Google Sign-In only. ​Logic: 30s loop generating 1.20x–4.50x multipliers with live history. ​Monetization: Pay-to-unlock via Flutterwave (Africa) and PayPal (Global). Store "Premium" status in Firestore. ​UI: Neon Green/Charcoal terminal aesthetic.
